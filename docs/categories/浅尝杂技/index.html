Github + Hugo 搭建博客(macos)

1 min read

Hugo

# github 配置 1. 新建仓库: killliu.github.io 2. 点击新仓库的 Settings 页签中的 Pages，设置自定义域名：note.killliu.com -> save # 配置域名DNS -> DNS 管理配置中增加一条记录 CNAME note killliu.github.io # 安装 hugo brew install hugo hugo version # 拉取仓库到本地 git clone https://github.com/killliu/killliu.github.io.git # cd 到该目录内，并创建 hugo 项目 app 并将其目录内的所有文件移动到上一层，删除 app 目录 hugo new site app # 添加一个主题 git clone https://github.com/spf13/hyde.git themes/hyde # 修改配置文件以指定主题 echo 'theme = "hyde"' >> config.toml ######## 添加文章 ######## hugo new posts/my-first-post.
